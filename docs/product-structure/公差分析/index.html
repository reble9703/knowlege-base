公差分析

公差分析是确保产品装配质量和功能的重要环节，合理的公差设计可以平衡产品性能和制造成本。

公差基础概念​

什么是公差？​

公差是允许尺寸变化的范围，是制造过程中不可避免的误差范围。

公差术语​

  • 基本尺寸 - 设计给定的理想尺寸
  • 实际尺寸 - 测量得到的尺寸
  • 极限尺寸 - 允许的最大和最小尺寸
  • 公差带 - 极限尺寸之间的范围
  • 配合 - 孔与轴之间的关系

配合类型​

间隙配合​

  • 孔的尺寸始终大于轴的尺寸
  • 用于需要相对运动的场合
  • 示例：轴承、滑动导轨

过渡配合​

  • 可能有间隙也可能有过盈
  • 用于定位要求较高的场合
  • 示例：销钉定位、轴承座

过盈配合​

  • 孔的尺寸始终小于轴的尺寸
  • 用于传递扭矩或固定连接
  • 示例：压入装配、热套

公差分析方法​

极值法（Worst Case）​

  • 考虑所有尺寸同时处于极限偏差
  • 计算简单，结果保守
  • 适用于装配要求高的场合

统计法（Statistical）​

  • 考虑尺寸分布的统计规律
  • 更符合实际生产情况
  • 可以放大公差，降低成本

GD&T（几何尺寸与公差）​

形状公差​

  • 直线度
  • 平面度
  • 圆度
  • 圆柱度

方向公差​

  • 平行度
  • 垂直度
  • 倾斜度

位置公差​

  • 位置度
  • 同轴度
  • 对称度

跳动公差​

  • 圆跳动
  • 全跳动

公差设计原则​

  1. 经济原则 - 在满足功能的前提下尽量放宽公差
  2. 工艺原则 - 考虑加工工艺的经济性
  3. 测量原则 - 考虑检测的可行性和成本
  4. 装配原则 - 考虑装配的难易程度

常用公差等级​

公差等级适用场合加工方法
IT1-IT4量规、高精度配合研磨、珩磨
IT5-IT7精密配合、轴承精车、精铣
IT8-IT10一般机械配合车、铣、钻
IT11-IT16非配合尺寸冲压、铸造
